# React技术栈-服务端渲染(SSR)

使用React+React-Router+Express+Webpack+babel实现服务端渲染。

## 技术栈

* [x] React
* [x] React Router 5
* [x] Webpack
* [x] Babel
* [x] Express

## 环境

* [x] Node 8+
* [x] React 16.8.0+
* [x] babel 7+
* [x] webpack 4+

## 运行

```shell
npm run dev
```
